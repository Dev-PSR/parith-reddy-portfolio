# Parith Reddy Portfolio

Personal portfolio site built with React and Vite.

The site uses a terminal-style interface with command shortcuts for experience, projects, skills, education, contact details, and resume access.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates the production build.
- `npm run preview` serves the production build locally.
- `npm run lint` runs ESLint.

## Project Notes

- Main app logic lives in `src/App.jsx`.
- Command content and responses live in `src/terminalData.js`.
- Styling lives in `src/index.css`.
- Static resume file lives in `public/ParithReddyResume.pdf`.
